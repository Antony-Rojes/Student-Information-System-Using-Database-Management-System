<h2>Project Overview</h2>

Student Information System Using Database Management System (DBMS) is a software application designed to manage, store, and retrieve student-related information efficiently. This system helps educational institutions maintain accurate records of students, courses, faculty, exams, and results using a structured database approach.

The project demonstrates practical usage of Database Management Systems along with front-end and backend integration concepts.

<h2>Objectives</h2>

Maintain student records in a centralized database

Manage courses, departments, faculty, and exams

Store and process student results efficiently

Reduce manual record-keeping errors

Provide quick data retrieval using SQL queries
